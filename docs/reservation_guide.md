# 福岡之旅 - 預約清單 & 攻略 (Reservation Guide)

這裡列出了行程中**必須預約**的項目，以及預約失敗時的 **Backup Plan (B 計畫)**。

---

## 🍽️ 餐廳預約 (Priority 1)

### **A. 鮨 品川 (Sushi Shinagawa)**
- **日期:** 12/15 (一) 晚餐
- **類別:** Omakase 壽司
- **重要性:** ⭐⭐⭐⭐⭐ (座位極少)
- **預約方法:**
  - 電話預約 (建議請信用卡白金秘書或酒店禮賓部幫忙)。
  - 嘗試 `TableCheck` 或 `Omakase.in` (如果有上架)。
- **Backup Plan:**
  - **鮨 山中 (Yamanaka) 本店:** 福岡米芝蓮名店，位於渡邊通。
  - **鮨 料理 一高:** 博多區的高級壽司，可用 TableCheck 預約。

### **B. 燒肉大喜 (Yakiniku Taiki)**
- **日期:** 12/18 (四) 晚餐
- **類別:** 高人氣燒肉 (厚切牛舌)
- **重要性:** ⭐⭐⭐⭐
- **預約方法:**
  - 電話 (日文)。
  - `AutoReserve` (可能有手續費)。
- **Backup Plan:**
  - **Yakiniku Nikuichi (肉いち) 博多店:** 極受遊客歡迎，可用 Google Maps / TableCheck 線上預約。
  - **燒肉 多牛 (Tagyu):** 性價比高，需現場排隊 (建議下午 3-4 點去拿籌)。

### **C. Princess Phi Phi (門司港)**
- **日期:** 12/15 (一) 午餐/下午
- **類別:** 燒咖哩名店
- **重要性:** ⭐⭐⭐
- **預約方法:**
  - HotPepper / Tabelog 預約「套餐 (Course)」。
- **備註:** 如果不想排隊，這是門司港少數可預約的名店。

### **D. Kawaya Gion (かわ屋 祇園店)**
- **日期:** 12/14 (日) 晚上
- **類別:** 博多名物 - 雞皮串燒
- **重要性:** ⭐⭐⭐⭐ (名店，經常客滿)
- **預約方法:**
  - 電話預約。
- **Backup Plan:**
  - **Torikawa Suikyou (鳥皮 粋恭):** 另一家雞皮名店，位於藥院。
  - **Torikawa Daijin (博多とりかわ大臣):** 有多間分店 (如博多站前)。

### **D. Nico Appartement**
- **日期:** 12/17 (三) 晚餐
- **類別:** 法式小酒館
- **重要性:** ⭐⭐⭐
- **預約方法:**
  - 電話 或 Instagram DM 詢問。
- **Backup Plan:**
  - **Yorgo:** 位於警固 (Kego)，需預約。
  - **Goh:** 亞洲 50 最佳餐廳之一。
